# Machine Learning en Economía 📈🤖

Este repositorio está dedicado al desarrollo de modelos de **Machine Learning aplicados al entorno económico**, sirviendo además como una guía con recursos de aprendizaje prácticos. 

Aquí encontrarás cuadernos de Jupyter (`.ipynb`) enfocados principalmente en **modelos de regresión** para el análisis y predicción de variables financieras y macroeconómicas.

### 🛠️ Bibliotecas Utilizadas
Para el desarrollo de estos proyectos se emplean las herramientas estándar del ecosistema de Data Science en Python:
* **Manipulación de Datos:** `pandas`, `numpy`
* **Modelado Clínico y ML:** `scikit-learn`, `statsmodels`
* **Visualización:** `matplotlib`, `seaborn`
